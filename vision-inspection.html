Overview

The system equipped with a robotic arm having 6 degree of freedom (DOF), by which it attacted with vision camera and move according to a predetermined point as per the production system requirement. A camera with the computer vision software detects various defects and dimension. Image Proccessing and machine learning has been used to determine the defects of the products. This multi-DOF robotic can be a remarkably useful tool for automating the production process completely. In the system, the efficiency of defects detection is around 99%, which proves the efficiency of the system. The overall improvement in the efficiency of the production process can be significantly enhanced by using this system.
